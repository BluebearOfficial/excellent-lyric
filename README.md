### 7. 饼说

古之冬季必有饼，饼者，所以传冷受寒解热也。今非大冰河期者 ，孰能无热 ？ 热而无寒潮，其为热也，终不解矣。生于吾西，其低温也固入川盆，吾观而盼之；生于吾东，其雪线也亦下岭南，吾观而盼之。吾盼冷也，夫庸知其路之西东于吾乎？是故无西无东，无北无南，冷之所存，饼之所存也。
嗟乎！冷冬之不在也久矣！欲人之无热也难矣！古之苦寒，其冷今也远矣，犹且盼冷而感囍；今之暴热，其热古代也亦远矣，而耻盼寒潮。是故冷益冷，热益热。严寒之所以为寒，爆热之所以为热，其皆出于此乎？爱其地，择站而迁之；于其身也，则耻冷焉，热矣。彼假冷之站，除之云而令其辐射者，非吾所谓传其冷解其热者也。辐射之皮鸭，热之不解，或饼焉，或否焉，小冷而大热，吾未见其寒也。南宁广州百越之地，不耻相冷，四川之盆地，曰冷曰积雪云者，则群鸭而笑之。问之，则曰:“彼与彼纬相若也，气相似也，位低而下沉，山高则逆温。”呜呼！严寒之不复，可知矣。南宁广州百越之地，川盆不齿，今其寒乃反不能及，其可怪也欤！
……
藏聂拉木，纬二八，好降雪，五囍丰登皆通享之，不拘于时，盼寒潮。余嘉其能行古道，作《饼说》以贻之。
