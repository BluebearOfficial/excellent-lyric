### 7. 饼说

古之冬季必有饼，饼者，所以传冷受寒解热也。今非大冰河期者 ，孰能无热 ？ 热而无寒潮，其为热也，终不解矣。生于吾西，其低温也固入川盆，吾观而盼之；生于吾东，其雪线也亦下岭南，吾观而盼之。吾盼冷也，夫庸知其路之西东于吾乎？是故无西无东，无北无南，冷之所存，饼之所存也。
嗟乎！冷冬之不在也久矣！欲人之无热也难矣！古之苦寒，其冷今也远矣，犹且盼冷而感囍；今之暴热，其热古代也亦远矣，而耻盼寒潮。是故冷益冷，热益热。严寒之所以为寒，爆热之所以为热，其皆出于此乎？爱其地，择站而迁之；于其身也，则耻冷焉，热矣。彼假冷之站，除之云而令其辐射者，非吾所谓传其冷解其热者也。辐射之皮鸭，热之不解，或饼焉，或否焉，小冷而大热，吾未见其寒也。南宁广州百越之地，不耻相冷，四川之盆地，曰冷曰积雪云者，则群鸭而笑之。问之，则曰:“彼与彼纬相若也，气相似也，位低而下沉，山高则逆温。”呜呼！严寒之不复，可知矣。南宁广州百越之地，川盆不齿，今其寒乃反不能及，其可怪也欤！
……
藏聂拉木，纬二八，好降雪，五囍丰登皆通享之，不拘于时，盼寒潮。余嘉其能行古道，作《饼说》以贻之。

### 13. 当我真正开始爱自己 

作者：卓别林

当我真正开始爱自己，
我才认识到，所有的痛苦和情感的折磨，
都只是提醒我：活着，不要违背自己的本心。
今天我明白了，这叫做“真实”。
当我真正开始爱自己，
我才懂得，把自己的愿望强加于人，
是多么的无礼，就算我知道，时机并不成熟，
那人也还没有做好准备，
就算那个人就是我自己。
今天我明白了，这叫做“尊重”。
当我开始爱自己，
我不再渴求不同的人生，
我知道任何发生在我身边的事情，
都是对我成长的邀请。
如今，我称之为“成熟”。
当我开始真正爱自己，
我才明白，我其实一直都在正确的时间，
正确的地方，发生的一切都恰如其分。
由此我得以平静。
今天我明白了，这叫做“自信”。
当我真正开始爱自己，
我不再牺牲自己的自由时间，
不再去勾画什么宏伟的明天。
今天我只做有趣和快乐的事，
做自己热爱，让心欢喜的事，
用我的方式、我的韵律。
今天我明白了，这叫做“单纯”。
当我开始真正爱自己，
我开始远离一切不健康的东西。
不论是饮食和人物，还是事情和环境，
我远离一切让我远离本真的东西。
从前我把这叫做“追求健康的自私自利”，
但今天我明白了，这是“自爱”。
当我开始真正爱自己，
我不再总想着要永远正确，不犯错误。
我今天明白了，这叫做“谦逊”。
当我开始真正爱自己，
我不再继续沉溺于过去，
也不再为明天而忧虑，
现在我只活在一切正在发生的当下，
今天，我活在此时此地，
如此日复一日。这就叫“完美”。
当我开始真正爱自己，
我明白，我的思虑让我变得贫乏和病态，
但当我唤起了心灵的力量，
理智就变成了一个重要的伙伴，
这种组合我称之为，“心的智慧”。
我们无须再害怕自己和他人的分歧，

形成新的世界，今天我明白，这就是“生命”。

### 15. 《起风了》重庆气象版

一八九月，秋雨漫天，从此关注气象超三年。

重庆天气，变化万千，多彩迷人眼。

十月秋风，感觉冷炸，入冬路顺滑。

一月很皮鸭，四月三十五，假入夏甚炎暑。

五月冷气阴雨连，气温大减，看着渝雪似在眼前，也初识城口高山地区的严寒。

七月份刷了凉夏，无40+。

八月锋前增温爆发，十月底的冷雨将气温压。

我曾感受一九二零烫冬，也记三月冰雹摸空。

我白熬夜，硝酸拿捏，笑容不绝。

我感慨沙坪坝的皮鸭，升破20三月十八。

这年四月冷气还有一些，在低温晴夜，赏天上皓月。

八月暴热，均温三十，整月降水竟然只有十。

成都暴雨六百毫米，重庆洪水积。

九月暴热冲三十八，却无高温假。

国庆秋雨寒，渝州升温难，物品难弄干。

十一月上中晴朗，秋高气爽，午后日照体感不凉，下旬冷气南下导致气温突降。

传说冷冬拉尼娜，寒潮南下，重庆西部却得皮鸭，盼望天空中飘落下雪花。

我曾听闻青杠操场结冰，也记沙坪坝未破零，城市中心，热岛效应，低温不行。

我难忘二月下旬热炸，当时我仍在放寒假，太阳毒辣，高温近二十八。

我还记七月青杠雷暴，可观降水，陪我入校。

暑假补课，气温特高，真是苦恼。

二一年的八月刷四十一，却现秋日在那月底，十九点九，我会永铭记。

我惊于九月满贯夏日，也叹整月不破二十。

进入十月，开头高温，后强冷气。

我曾感慨于漫天雪花，痴迷于你的春秋冬夏。

三年有你，我很开心啊。再追三年，我也愿意啊。
